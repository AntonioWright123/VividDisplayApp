# VividDisplayApp — Real-Time Client Photo Display

## 📸 Project Description

VividDisplayApp enables real-time display of tethered camera photos, perfect for live client sessions.  
When a photo is taken, it instantly appears in the gallery, allowing clients to preview images seconds after capture.

Designed for **Vivid Snaps Photography** and customizable for other studios.

---

## ✨ Features

- ✅ **Live Photo Display**: Instantly updates with the newest photo as it's captured.
- ✅ **Automatic Folder Watching**: Monitors a selected session folder without manual refresh.
- ✅ **Favorites Directory Creation**: Automatically prepares a favorites folder for each session.
- 🛠 **Branded Layout** (planned): Customizable with studio logos and themes.
- 🛠 **Photo Counter** (planned): Track number of images captured.
- 🛠 **Favorite Selection by Clients** (planned): Clients can pick favorite images during the session.
- 🛠 **Full Gallery Grid View** (planned): Browse all images easily.
